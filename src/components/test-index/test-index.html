컴포넌트 소개

우리는 쥬니버의 컴포넌트를 직접 구현하며 사이트를 세밀하게 재현해 보았어요.
여백과 구조를 조정하고, 시멘틱한 태그를 적용해 더 나은 형태로 구성했어요.

단순한 복제가 아니라, 협력하며 쌓아가는 과정에서 많은 것을 배울 수 있었어요.
아직 부족한 점이 있지만, 성장하는 과정이 더 소중하니까요!
그럼, 우리가 만든 컴포넌트들 직접 슬라이드해서 확인해 보세요!
