## 📝 프로젝트 개요
2025년 5월 27일 쥬니어네이버 서비스 종료에 아쉬운 마음을 담아, 멋쟁이사자처럼 프론트엔드 부트캠프 4조에서 개발한 학습용 UI 프로젝트입니다. 상업적인 목적이 아닌 학습을 위한 프로젝트로, 프론트엔드 기술을 실습하며 경험을 쌓고자 했습니다.
